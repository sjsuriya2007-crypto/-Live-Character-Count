# ✍️ Live Character Count

## Overview

**Live Character Count** is a simple and interactive web application built using **HTML, CSS, and JavaScript**. It displays the number of characters entered into a text area in real time, providing instant feedback as the user types. This project is useful for understanding DOM manipulation, event handling, and dynamic UI updates.

## Features

* Real-time character counting
* Updates instantly while typing
* Displays the current number of characters entered
* Handles empty input correctly
* Clean and responsive user interface
* Beginner-friendly JavaScript project

## Technologies Used

* HTML5
* CSS3

## How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Double-click **index.html** or open it in any modern web browser.
4. Start typing in the text area to see the live character count update automatically.

## How It Works

* The application listens for keyboard input in the text area.
* Every time the user types or deletes a character, JavaScript updates the displayed count instantly.
* The character counter always reflects the current length of the entered text.

## Expected Output

As the user types in the text area, the displayed character count increases or decreases in real time. The counter updates immediately without requiring the page to refresh, providing a smooth and interactive user experience.
